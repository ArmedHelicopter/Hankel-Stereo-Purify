# 第 0 章：架构总览（一页纸）

## 核心链路：采样如何被「变形」

```mermaid
flowchart TB
  subgraph io [I_O]
    blocks["blocks解码 float64"]
  end
  subgraph frame [单帧_OLA]
    stereo["stereo F×2"]
    A["A Hankel 视图 L×K×2"]
    B["B 联合 L×2K"]
    C["C SVD 截断"]
    D["D 对角平均 N×2"]
    stereo --> A
    A --> B
    B --> C
    C --> D
  end
  subgraph macro [整文件]
    ola["帧起点列表 + Hanning"]
    acc["累加器 / 可选 memmap"]
    write["soundfile.write"]
  end
  blocks --> stereo
  D --> ola
  ola --> acc
  acc --> write
```

**一句话**：磁盘上是流；进入算子前被切成 **OLA 帧**；每帧在核心四层里走 **波形 → Hankel 视图 → 联合块 → 低秩块 → 对角平均回波形**；多帧重叠相加后写回文件。

### 最小符号表（抄笔记）

| 符号 | 含义 |
|------|------|
| \(F\) | 单帧样本数（每声道 \(F\) 点） |
| \(L\) | Hankel 窗口长（行数） |
| \(K\) | \(K=F-L+1\)（单声道 Hankel 列数） |
| 联合块 | 形状 **\(L\times 2K\)**（左右各 \(K\) 列） |
| D 输出长度 | 单声道对角平均序列长 **\(L+K-1\)**；立体声为 `(N,2)`，其中 \(N=L+K-1\)，详见 [04](04_对角平均与重构.md) |

### 单帧类型流（函数链上刻意不一致）

单帧入口为 **`FloatArray` 形状 `(F,2)`**。链内：`hankel_embed` 输出 **`tuple`**（左右各一块）；`combine_hankel_blocks` 起为二维 **`FloatArray`** 联合块；`make_svd_step` 闭包与 `diagonal_reconstruct` 延续矩阵与重构帧类型。共享类型别名见 [`src/core/array_types.py`](../src/core/array_types.py)；实现见第 [01](01_从波形到矩阵.md)、[02](02_立体声联合块.md) 章代码锚点。

## 工程亮点（工业级痕迹）

| 机制 | 位置（示意） | 目的 |
|------|----------------|------|
| Hankel 零拷贝 | `a_hankel.py` `as_strided` | 避免 Python 嵌套循环填矩阵 |
| 联合块连续内存 | `b_multichannel.py` `ascontiguousarray` | 喂 LAPACK/BLAS |
| 截断 SVD / 全谱 | `c_svd.py` `svds` vs `linalg.svd` | 固定秩省算力；能量模式需全谱 |
| W-corr 首帧冻结（能量） | `c_svd.py` `_energy_w_corr_frozen` | 避免每帧 k 变就重算整张 W |
| 反对角聚合 | `d_diagonal.py`：预计算 `t_flat=i+j`，`numpy.bincount` 按对角求和；batch 维小循环（如立体声 `B=2`） | 避免 `O(B·mn)` 级 scatter 整型索引表；控制峰值分配 |
| 有界队列 + 毒丸 | `pcm_producer.py` | 背压与优雅退出 |
| 整文件 OLA + memmap 写回 | `soundfile_ola.py`（`SoundfileOlaEngine`）；`purifier.py` 为入口与 `process_file` | 门面拆分：单帧数学在 `core/`，整文件调度在 facade |
| 路径/后缀白名单 | `audio_formats.py` | 防御性 I/O |
| 分层异常 + 可机读包装类型 | `exceptions.py`：`ProcessingError.origin_exception_type`（`module.QualName`）；门面 `process_file` 填充 | 粗分桶文案不变，排障可区分 `MemoryError` / `KeyError` 等 |

## 知识挂钩速查

| 现象 | 线性代数 | 软件工程 |
|------|----------|----------|
| Hankel 块 | 滞后嵌入、秩与轨迹矩阵 | 视图与所有权 |
| 联合块 SVD | 主子空间、奇异值能量 | 策略模式、固定秩 vs 能量 |
| 对角平均 | 逆嵌入、一般非 Hankel | 算法闭环与近似 |
| OLA | 短时平稳假设 | 分帧、窗、重叠 |
