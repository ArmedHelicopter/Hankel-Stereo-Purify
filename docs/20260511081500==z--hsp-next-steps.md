* 第一梯队：理论根基 + 边界探索（并行启动）

**P1-A：深挖 MSSA 各环节（方向四）**

数学本质：知道"怎么做有效"不等于知道"为什么有效"。

阶段 A — Hankel 嵌入：
- [ ] L 在子空间域控制什么？频率分辨率 vs 时间分辨率权衡
- [ ] K=F-L+1 影响秩的估计精度
- [ ] 音频 Hankel 矩阵的秩分布规律：top-16 = 99.8% 是否普遍？
- [ ] L 的理论最优值？连接 Prony/ESPRIT 子空间维数估计
- [ ] 优势放大：零参数基变换，信号结构自动决定子空间

阶段 B — 立体声联合块：
- [ ] 联合块 vs 独立处理的相位差量化
- [ ] 联合块的秩 vs 各声道秩之和
- [ ] 相位锁定失效条件
- [ ] 优势放大：相位保持是 DL 的结构性弱点

阶段 C — SVD 截断：
- [ ] 奇异值的物理含义：σ₁/σ₂ 各对应什么
- [ ] 固定秩 vs 能量阈值 vs Wiener 的本质区别
- [ ] Wiener 中 σ_noise 的估计方法
- [ ] 优势放大：数据驱动的自适应滤波

阶段 D — 对角平均：
- [ ] 对角平均丢失了什么信息
- [ ] 方差稳定性
- [ ] 优势放大：隐式正则化，强制局部自相关结构

BPW 预处理：
- [ ] 白化为什么有效？形式化证明条件数改善
- [ ] α=0.75 比 α=1.0 好的物理含义
- [ ] 白化 + MSSA 是否等价于频率相关权重的 SVD
- [ ] 优势放大：频域先验与子空间方法的结合

**P1-B：失败模式分析（方向五·5.1）**

系统性找边界：
- [ ] 瞬态信号测试：鼓点、钢琴 attack → rank 急剧变化时截断是否抹掉瞬态
- [ ] 左右声道不相关测试：伪立体声 → 联合块假设崩溃条件
- [ ] 非平稳噪声测试：风声、人声背景 → bandpass 2kHz 硬分频失效边界
- [ ] 低 SNR 极端测试：噪声 > 信号 → 低秩假设破坏阈值
- [ ] 输出：每种失败模式的触发条件、症状、缓解方案

**P1-C：Prony/ESPRIT 理论连接（方向五·5.5）**

**P4-B：跨领域泛化（方向六）**

核心洞察：Hankel → SVD → 对角平均是纯数学操作，带通滤波器是唯一跟音频强耦合的模块。

| 领域 | 噪声特征 | 预处理替代 |
|------|---------|-----------|
| 地震学 | 宽带随机噪声 | 直接 MSSA |
| 机械振动 | 周期性冲击 + 宽带 | 带通保留故障频率 |
| ECG/EEG | 工频 + 肌电 | 陷波滤波器 |
| 金融 | 噪声交易 + 信息信号 | 低通或无预处理 |
| 引力波 | 探测器噪声 | 探测器噪声谱白化 |
| 雷达/声纳 | 杂波 + 热噪声 | 杂波对消 |

待做：
- [ ] 将管线抽象为"核心不变 + 预处理可插拔"的策略模式
- [ ] ECG/EEG 作为第一个非音频测试
- [ ] 金融时间序列作为第二个非音频测试

**P4-C：MSSA → DL 数据管线（HSP 为深度学习生产 AI-ready 数据集）**

核心洞察：DL 降噪模型需要 clean/noisy 配对数据训练，但真实世界中干净的参考信号极稀缺。MSSA 可以从纯噪声录音中提取出"足够干净"的伪参考信号，为 DL 训练提供数据源。

这个方向把 HSP 的定位从"DL 的竞争者"变成"DL 的上游"——不是替代深度学习，而是让深度学习能用更多数据。

**首要落地场景：EEG 信号**

思路：MSSA 做降噪前端 → DL 做解码后端（脑机接口、癫痫检测、情绪识别等）。

优势：
- EEG 噪声特征明确（工频 50/60Hz + 肌电 + 眼电），bandpass 替换成陷波即可
- DL 不需要同时学降噪+解码，任务简化，训练数据需求大幅降低
- MSSA 的确定性 + 可逆性满足医疗场景的可解释性要求
- 数据易获取（公开 EEG 数据集多），下游任务明确

待做：
- [ ] EEG 公开数据集调研：PhysioNet、BCI Competition、TUH EEG Corpus
- [ ] MSSA 降噪后 EEG 的信噪比提升量化
- [ ] 对比：端到端 DL vs MSSA降噪+DL解码 的下游任务性能差异
- [ ] 量化：MSSA 伪标签引入的偏差有多大？→ 用"偏差上限"来评估 DL 模型的理论性能天花板
- [ ] 优势：MSSA 无训练数据依赖 + 可逆性（可审计伪标签质量）→ 比 DL 做伪标签更可靠
- [ ] 风险：MSSA 本身的截断误差会传播到 DL 模型中，需要评估"误差传播链"

**P4-D：反向应用——噪声提取**

核心洞察：MSSA 管线（Hankel → SVD → 截断 → 重建）的本质是"分离信号子空间和噪声子空间"。正向用是去噪，反向用就是提取噪声——把截断掉的部分重建出来，就是纯噪声信号。

优势：
- 提取的噪声是真实的、结构化的，不是人工合成的白噪声
- 可以分离出"不同类型的噪声"：低秩部分 = 信号，高秩残差 = 噪声
- 对于非平稳噪声（风声、人声背景、机械干扰），提取出来的噪声 profile 可以用于后续的自适应降噪

应用场景：
- 噪声轮廓估计（noise profile estimation）：提取出噪声后，可以精确建模噪声的频谱特征，反过来指导降噪参数选择
- 噪声消除耳机/ANC：实时提取环境噪声，生成反相声波
- 音频取证：从录音中分离出背景噪声，分析录音环境
- 音效设计：从自然录音中提取干净的噪声素材（雨声、风声、机械声）
- 数据增强：提取真实噪声 profile 后，可以叠加到干净信号上生成训练数据

待做：
- [ ] 验证：残差信号（original - denoised）的噪声纯度——是否真的只有噪声，还是混入了信号成分
- [ ] 不同 energy_fraction 截断下，残差信号的频谱特征变化
- [ ] 噪声提取的可逆性：denoise(signal) + denoise(noise) ≈ original？
